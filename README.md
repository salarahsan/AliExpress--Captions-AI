🚀 AliExpress AI Caption Pro
AliExpress AI Caption Pro aik powerful aur lightweight Chrome Extension hai jo affiliate marketers ke liye design ki gayi hai. Ye Groq AI (Llama 3) ka use karte hue kisi bhi AliExpress product title se viral aur high-converting social media captions generate karti hai.

✨ Features
AI-Powered Captions: Product titles se foran engaging captions banayein.

Optimized for Sales: Captions mein auto-generated Hooks, Bullet points, aur Call-to-Actions (CTA) shamil hain.

Privacy Focused: Aapki API Key aapke browser mein locally save rehti hai (Safe & Secure).

Clean UI: Modern aur user-friendly interface.

Lightning Fast: Groq API ki badolat captions kuch hi seconds mein tayyar.

🛠️ Tech Stack
Frontend: HTML5, CSS3

Logic: JavaScript (Chrome Extension API)

AI Model: Llama 3.3 via Groq Cloud API

🚀 How to Install (Manual Mode)
Kyunke ye extension abhi development stage mein hai, aap isay manually install kar sakte hain:

Is repository ko Download ZIP karke extract karein ya git clone karein.

Chrome browser open karein aur chrome://extensions/ par jayein.

Top-right corner se Developer Mode ko ON karein.

Load unpacked button par click karein aur is project ka folder select karein.

Extension icon ko browser bar mein Pin kar lein.

🔑 How to Use
Apni Groq API Key hasil karein.

Extension kholein aur apni API Key enter karein (ye sirf aik baar karni hogi).

AliExpress se kisi bhi product ka title copy karke box mein paste karein.

"Create Viral Caption" par click karein aur apna result hasil karein!
